# Gulfshine Services Inc. / Wingolf Travel Website Redesign

## Goal

Redesign the existing travel website into a modern, premium, easy-to-maintain static HTML/CSS/JS site.

The site should feel:
- luxury travel
- cinematic
- clean
- trustworthy
- easy to browse
- mobile-first

The site should NOT feel:
- cluttered
- outdated
- like a cheap tourism template
- like hundreds of disconnected pages

## Brand Direction

Main brand: Gulfshine Services Inc. / Wingolf Travel

Core positioning:

Luxury travel experiences with two main paths:

1. Visit Argentina
2. Golf Travel

A third homepage area should show general destinations:
- beaches
- Caribbean
- Europe
- USA
- other international golf/travel destinations

## Required Legal Text

Add this phrase to all website pages and advertising areas:

“Fla. Seller of Travel Ref. No. ST46553.”

Place it in the footer on every page.

## Logo

Use the attached Gulfshine Services Inc. logo.

Logo style:
- premium
- colorful
- golf + travel identity
- should be used clearly in the header and footer

## Main Website Structure

### Home
Purpose: Give users two clear paths while showing the broader travel offering.

Sections:
1. Header/navigation
2. Hero section with luxury travel message
3. Two main feature panels:
   - Visit Argentina
   - Golf
4. General destinations section:
   - Caribbean
   - Beaches
   - Europe
   - USA
   - Rest of the World
5. Featured trips/cards
6. Why travel with us
7. Inquiry CTA
8. Footer with Seller of Travel phrase

### Visit Argentina
Purpose: Showcase Argentina and Patagonia travel experiences.

Sections:
- Hero image of Argentina/Patagonia
- Destination cards:
  - Buenos Aires
  - Iguazú
  - Bariloche
  - El Calafate
  - Ushuaia
  - Mendoza
  - Puerto Madryn
  - Salta
- Featured itineraries
- Custom trip CTA

### Golf
Purpose: Showcase golf travel packages.

Sections:
- Hero image of golf travel
- Golf destination cards:
  - Argentina
  - Caribbean
  - USA
  - Scotland
  - Ireland
  - Spain
  - Portugal
  - Italy
- Featured golf packages
- Custom golf trip CTA

### Journeys
Purpose: Browse trip packages.

Features:
- Search bar
- Filter buttons:
  - Argentina
  - Golf
  - Caribbean
  - Europe
  - USA
- Large visual cards
- Each card should show:
  - destination
  - short description
  - trip length if available
  - CTA button

### Contact / Request Pricing
Purpose: Lead generation.

Fields:
- name
- email
- phone
- destination interest
- trip type
- message

CTA text:
“Plan Your Trip”

## Visual Style

Use:
- large cinematic imagery
- elegant serif headings
- clean sans-serif body text
- dark overlays on hero images
- warm gold accents
- deep green / navy / charcoal backgrounds
- lots of whitespace
- large cards, not tiny grids

Suggested colors:
- charcoal: #111111
- deep green: #062f2d
- gold: #c9972b
- cream: #f7f2e8
- white: #ffffff

Typography:
- headings: elegant serif
- body: clean sans-serif

Design references:
- Aman Resorts
- Black Tomato
- Pelorus
- high-end golf travel brands

## UX Rules

- Keep navigation simple.
- Do not show every destination at once on the homepage.
- Use cards and filters to organize content.
- Mobile must look polished.
- Avoid long walls of text.
- Use old site content/images as reference, but rewrite and simplify.

## Technical Rules

Build as static files only:

- index.html
- visit-argentina.html
- golf.html
- journeys.html
- contact.html
- css/style.css
- js/main.js
- assets/images/

No:
- WordPress
- React
- database
- build tools
- frameworks

Must be easy to upload directly into public_html.

## Content Source

- golf 2

assets/images/

Do not edit the old folders.
